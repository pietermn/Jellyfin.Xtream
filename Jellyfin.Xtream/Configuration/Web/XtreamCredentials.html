Apply regex replacements to Live TV, movie, series, season, episode, and category names. Use one rule per line: regex => replacement. Omit the replacement to remove matches. Lines starting with # are ignored. Existing unscoped rules continue to apply everywhere. Optionally prefix a rule with scopes, for example [LiveChannel,Filesystem] ^NL\s*-\s* => . Available scopes: LiveChannel, LiveProgram, Category, Vod, Series, Season, Episode, and Filesystem. Invalid or slow rules are skipped and reported in the server log.
